1.Python 的异步模块 asyncio 的概念和基本用法

从Python3.4开始引入asyncio模块，增加了异步编程。asyncio最大的特点就是只存在一个线程，采用“多任务合作模式”，允许异步任务交出执行权给其他任务，等到其他任务完成，再收回执行权往下执行。

例子:

#!/usr/bin/env python
# async.py

import asyncio

async def count():
	print("One")
	await asyncio.sleep(1)
	print("Two")

async def main():
	await asyncio.gather(count(),count(),count())

asyncio.run(main())

上面的脚本中，在async函数main内，asyncio.gather()方法将多个异步任务包装成一个新的异步任务，必须等到内部的多个异步任务都执行结束，这个新的异步任务才会结束。

脚本运行结果:

$ python3 async.py
One
One
One
Two
Two
Two

实例:pyppeteer模块

import asyncio
from pyppeteer import launch


async def main():
    browser = await launch()
    page = await browser.newPage()
    await page.goto('https://www.bilibili.com')
    await page.screenshot({'path': 'bili.png'})
    await browser.close()

# python3.7
# asyncio.run(main())

# python3.5~3.6
loop = asyncio.get_event_loop()
loop.run_until_complete(asyncio.wait([main()]))

需要注意的地方是python3.5~3.6中执行以上代码会遇到module 'asyncio' has no attribute 'run',详情见参考链接。

1.分享一个背单词网站，从出现次数最多的单词开始背起

刷B站的时候看到这个视频，瞬间就来了兴趣，这个背单词网站是B站一位UP主为了提高自己英语听力和阅读能力为自己开发的一款基于Web的英语学习软件，经过内测之后现在免费开放给互联网的广大朋友们。

网站主要分为两部分，分别为记单词和听力练习部分。

单词部分

目前单词部分已经导入的词库有大学英语，出国考试，中学英语，专四专八，程序员英语，所有单词基于真题文本或者相关的阅读材料，利用程序统计词频，排除掉垃圾词，从词频最高的单词开始背起。目前单词部分最有意思的功能（上传个人词典）因为涉及到网站安全问题目前暂未上线。

目前网站采取的复习方式是：每10个单词一组，当用户点击太简单了，这个单词就不再安排复习，当用户点击基本认识或者提示一下，后台会根据相关算法多次安排复习，每组学习会安排20个单词复习，10个单词学习。在一本单词背完了之后再通读原材料复习复习。

听力部分

听力部分的材料也比较多，比如：英语四六级，雅思官方真题，TED，程序员英语等等英语材料，采取听力内容+字幕的形式，逐句精听。

官方建议的食用方式：

  • 第一遍的时候从头到尾去听，不要去看字幕，不要点击【这句听不懂】按钮；
  • 然后第二遍的时候，再从头到尾听，然后遇到第一遍没听懂的这个句子，你点一下【这句听不懂】按钮，但是这时候也不要暂停，还是一路从头走到尾，把这段里边所有的没听懂的句子都点完了【这句听不懂】之后；
  • 然后第三遍，看每一句【这句听不懂】的原文，这个原文里边不懂的单词、不懂的语法，你就去查，搞明白了之后，然后就闭着眼不看字幕，把一句话反复去听，直到能够听懂为止，再听下一句话；
  • 等所有的句子都听懂了之后，然后从头到尾再不看字幕、不暂停的把音频听一遍，如果发现还是有不懂的再重复以上步骤，直到这一段音频能够懂为止。这个过程会很漫长，但是只要坚持下来，进步非常明显。

挺有意思的一个网站，当然对于anki用户也可以参考参考，比如使用程序统计阅读材料的词频，调用词典接口生成例句等，然后调用ankiConnect生成anki卡片，或者导出为csv等anki能导入的格式。最后导入anki，于是就能愉快的玩耍了。
